# Brief del CRM para Productora

## Objetivo

Centralizar clientes, proyectos, presupuestos, tareas y comunicacion en un solo lugar.

## Problemas a resolver

- Informacion dispersa entre chats, planillas y notas.
- Falta de seguimiento de estados por proyecto.
- Dificultad para controlar cobros y entregas.

## Modulos iniciales (MVP)

- Clientes
- Proyectos
- Presupuestos
- Tareas internas
- Calendario de produccion

## Preguntas pendientes

- Tipos de productora y servicios exactos.
- Roles de usuario (admin, productor, editor, etc.).
- Flujo de aprobacion de presupuestos.
