# CRM Productora

Base de trabajo para definir y construir un CRM enfocado en productoras.

## Estructura

- `docs/`: definicion funcional y plan del producto.
- `db/`: esquema SQL, tablas y migraciones iniciales.
- `integraciones/`: configuraciones de servicios externos.
- `assets/`: logos, imagenes y recursos visuales.

## Siguientes pasos

1. Completar `docs/brief.md` con objetivos y alcance.
2. Definir entidades en `db/schema.sql`.
3. Confirmar integraciones reales en `integraciones/env.example`.
